Кыргызстан стал лучшей страной для путешествий — британский рейтинг


Вторую позицию после КР занимает Иран, третью — Грузия Кыргызстан в рейтинге туристического портала Британии Wanderlust стал лучшим в номинации "Развивающееся направление". В начале февраля Wanderlust опубликовал список лучших стран для посещения в 2018 году.


Кыргызстан вошел в топ-5 стран для туризма на 2018 год — британский рейтинг В публикации портала разъясняются причины для путешествия по стране, один из основных — через Кыргызстан пролегал Великий Шелковый путь. Вторую позицию в рейтинге занимает Иран, третью — Грузия.


Специально для путешественников у нас есть лонгрид, из которого можно узнать, как добраться до Кыргызстана, какие места в республике стоит обязательно посетить и почему поездка к нам в зимнее время ничуть не хуже путешествия летом.
